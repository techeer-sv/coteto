# 🐱 **coteto**
**코딩테스트 테토남/녀가 되기 위한 레포**

> _"코딩테스트 연습을 통해 모두 테토남·테토녀가 되자!"_

---

## ✨ coteto란?
**coteto**는 "**코딩테스트를 연습하는 테토남/테토녀**"의 줄임말입니다.  
여기서는 꾸준한 문제 풀이와 학습을 통해 누구나 코딩테스트 실력을 탄탄히 다지고,  
어떤 기업의 시험이 와도 당황하지 않는 **테토남·테토녀**를 목표로 합니다!

---

## 💡 왜 코딩테스트 연습을 해야 할까?

### 1. **대부분의 기업이 평가 기준으로 활용**
많은 기술 기업들이 실력 평가의 첫 단계로 코딩테스트를 사용합니다.  
좋은 점수 = 기회의 문이 더 활짝 열린다는 뜻이죠!

### 2. **논리적 사고력과 문제 해결 능력 향상**
문제를 구조적으로 분석하고 다양한 접근 방식으로 해결해 보며  
자연스럽게 개발자로서의 사고 체계가 단단해집니다.

### 3. **면접 대응 능력 상승**
CS 지식 + 알고리즘 사고력이 합쳐지면  
기술 면접에서도 훨씬 명확하고 설득력 있게 답변할 수 있습니다.

### 4. **성공 경험이 쌓이며 자신감 Up**
문제를 풀 때마다 지식이 쌓이고,  
난이도 높은 문제를 해결할 때마다 "나도 할 수 있다"는 성취감이 생깁니다.

---

## 🔥 테토남/테토녀가 되면 좋은 이유

### ⭐ 어떤 코딩 테스트도 두렵지 않다
정답이 틀리면 사실 내가 아니라 문제가 틀린게 아닐까요?

### ⭐ 기업들의 신뢰를 받는 실력자가 된다
기업은 자신있는 테토남/녀를 원하지 에겐을 원하지 않습니다. 기업에 매달리는게 아닌 기업이 매달리는 인재가 되어야합니다

### ⭐ 실제 개발 업무에서 문제해결할때도 빛을 발한다
회사에서 문제가 생겼을시 자신의 알고리즘 해결능력을 믿으면서 
"닥치고 내가 하자는대로 하자" 라고 할수있습니다.
